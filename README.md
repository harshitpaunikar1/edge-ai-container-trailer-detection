# Edge AI Container & Trailer Detection

> **Domain:** Logistics & Transportation

## Overview

Automated detection system for yard operations using computer vision. The solution processes camera feeds at the edge to identify containers and trailers, delivering structured events to downstream systems without expanding network infrastructure.

## Skills & Technologies

- Computer Vision
- Edge AI Deployment
- Object Detection
- Model Quantization
- Data Annotation
